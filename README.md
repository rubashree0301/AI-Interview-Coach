# AI-Interview-Coach

AI Interview Coach is a project that helps students practice technical interviews using AI.

## Features
- AI-based interview questions
- Instant feedback
- Coding practice
- Voice interaction

## Technologies Used
- Python
- NLP
- Machine Learning
- Streamlit / Flask

## Installation

1. Clone the repository
git clone https://github.com/rubashree0301/AI-Interview-Coach.git

2. Go to project folder
cd AI_Interview_Coach

3. Install dependencies
pip install -r requirements.txt

4. Run the project
python app.py
